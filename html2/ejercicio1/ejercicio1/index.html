>html

HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el código estándar que se utiliza para estructurar y desplegar el contenido de las páginas web. Piensa en él como el esqueleto o plano de una construcción que define dónde van los títulos, los párrafos, las imágenes y los enlaces

css

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es el lenguaje que define la apariencia visual de una página web. Si el HTML es el esqueleto, el CSS es la pintura, la ropa y el diseño que lo hacen lucir bien.

java

JavaScript (JS) es un lenguaje de programación ligero, interpretado y orientado a objetos, fundamental para el desarrollo web. Permite crear sitios web dinámicos e interactivos, añadiendo comportamiento como animaciones, actualizaciones de contenido sin recargar la página y validación de formularios directamente en el navegador del cliente.
