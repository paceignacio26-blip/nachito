HTML

(HyperText Markup Language) es el estándar utilizado para crear y estructurar el contenido de las páginas web. No es un lenguaje de programación, sino un "lenguaje" de marcado que define la estructura y el significado del contenido mediante etiquetas.

CSS

/p>parra(siglas de Cascading Style Sheets, en español Hojas de Estilo en Cascada) es un lenguaje de estilo utilizado para definir y controlar la presentación visual de documentos escritos en lenguajes de marcado como HTML o XML.
